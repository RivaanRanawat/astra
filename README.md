## Tech Stack

**Client:** Flutter

**Server:** Firebase, Hive, SQFLite, AgoraRTC

  
## Features

- Google Signin
- Searching User
- Chatting With Users
    - Image Sharing (From Gallery & Camera)
    - Text Sharing
    - Emojis
- Video Calling
